# üåê Web Developer Summer Intern

This repository hosts the code for a modern, fully responsive **Single-Page Application (SPA)** developed during a one-month industrial training internship. The project aimed to create a digital presence for Guruprabha Consultants Pvt. Ltd., enhancing client outreach and communication efficiency.

---

## ‚ú® Key Features

* **Responsive Design (UX/UI):** Built using a mobile-first approach and **CSS3 media queries** to ensure seamless compatibility across all devices (desktops, tablets, and smartphones).
* **SPA Architecture:** Structured as a Single-Page Application for faster load times and smooth transitions between sections (Home, About Us, Services, Contact Form).
* **Automated Inquiry System:** Integrated the **EmailJS API** to handle contact form submissions directly from the client side, eliminating the need for a complex backend server infrastructure.
* **Client-Side Validation:** **JavaScript** was used for real-time form validation, ensuring user inputs were correct before submission and improving overall usability.
* **Semantic HTML:** Utilizes **HTML5 semantic elements** (`<article>`, `<section>`, `<footer>`) to improve SEO and accessibility.

## üöÄ Getting Started

This project relies on standard web technologies (HTML, CSS, JS) and uses a package manager (Yarn/npm) for development dependencies.

### Prerequisites

* **Code Editor:** Visual Studio Code, Sublime Text, etc.
* **Browser:** Google Chrome, Firefox, or Edge.
* **Package Manager:** Yarn or npm (required to install development dependencies).

### Setup and Execution

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/the-rajanand/web-developer-summer-intern.git](https://github.com/the-rajanand/web-developer-summer-intern.git)
    cd web-developer-summer-intern
    ```

2.  **Install Dependencies:**
    Use the package manager to install local development dependencies (if any):
    ```bash
    npm install  # or yarn install
    ```

3.  **Run the Website:**
    The website can be executed by simply opening the main HTML file in any browser.
    * Open the root file (`main.html`) in your web browser.

4.  **Configure EmailJS:**
    To enable the contact form, you must configure the EmailJS service:
    * Locate the EmailJS integration code (likely in a separate `.js` file or the main HTML).
    * Replace the placeholder IDs (`YOUR_USER_ID`, `YOUR_SERVICE_ID`, `YOUR_TEMPLATE_ID`) with your own valid keys from the EmailJS dashboard.
